# دليل اختصارات لوحة المفاتيح

يساعد دليل الاختصارات التالي على التنقل وإنجاز المهام داخل منصة رابِط بسرعة أكبر. تدعم الاختصارات الأنظمة المختلفة (macOS و Windows / Linux)، ويمكن الوصول إلى القائمة الكاملة من داخل التطبيق عبر الضغط على `Shift + /` أو من لوحة الأوامر الجديدة عبر `⌘/Ctrl + K`.

| الاختصار | الوصف |
| --- | --- |
| `⌘/Ctrl + K` | فتح لوحة الأوامر والبحث السريع |
| `Shift + /` | عرض نافذة مرجع الاختصارات |
| `⌘/Ctrl + G` | الانتقال إلى لوحة التحكم للشركات |
| `⌘/Ctrl + T` | فتح مركز الأدوات (الحاسبات والمولدات) |
| `⌘/Ctrl + L` | التبديل بين العربية والإنجليزية |
| `⌘/Ctrl + .` | تبديل الوضع الداكن/الفاتح |
| `⌘/Ctrl + Alt + M` | تفعيل/إلغاء تقليل الحركات للمستخدمين الحسّاسين |

> **ملاحظات:**
> - جميع الأوامر متاحة أيضاً من لوحة الأوامر داخل التطبيق، مع وصف سريع وإمكانية البحث بالكلمات المفتاحية.
> - الاختصارات تتجاهل تلقائياً حقول الإدخال والنصوص القابلة للتحرير لمنع التعارض أثناء الكتابة.
> - يمكن توسيع هذه القائمة بإضافة أوامر جديدة داخل الملف `client/src/components/KeyboardShortcuts.tsx`.
