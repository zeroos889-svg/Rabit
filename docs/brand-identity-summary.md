# الهوية البصرية لمنصة رابِط

## الشعار

- **التصميم:** رابط مزدوج (سلسلتان متداخلتان) مع تدرج سماوي → أزرق ملكي وكلمة "رابط" و"Rabit" باللون الأزرق العميق
- **الملف:** `client/public/LOGO.svg`
- **الاستخدام:** متوازن على الخلفيات الفاتحة والداكنة مع نسخة SVG واحدة فائقة الدقة
- **الأحجام المتاحة:** 32px، 48px، 64px، 96px، 192px (من نفس ملف SVG)

## نظام الألوان الرئيسي

### الألوان الأساسية

- **Sky Link:** `#6AD4FF`
- **Azure Core:** `#4F9DFF`
- **Royal Chain:** `#1F53FF`
- **Indigo Wordmark:** `#14238A`

### التدرجات

1. **تدرج سلسلة رابط:** من `#6AD4FF` → `#4F9DFF` → `#1F53FF`
2. **تدرج الكلمة:** من `#143FCE` → `#092982`

### ألوان الفئات

- **الشركات:** تدرج أزرق (`#4F9DFF` → `#1D4ED8`)
- **مزودو الخدمات المستقلون:** تدرج بنفسجي (`#7C3AED` → `#5B21B6`)
- **الموظفون:** تدرج أخضر (`#10B981` → `#047857`)

### ألوان الحالات

- **نجاح:** `#10B981` (أخضر)
- **تحذير:** `#F59E0B` (برتقالي)
- **خطأ:** `#EF4444` (أحمر)

## الخطوط

- **العربية:** Cairo (300, 400, 500, 600, 700)
- **الإنجليزية:** Inter (300, 400, 500, 600, 700)

## CSS Classes المتاحة

- `.gradient-primary` - التدرج الأساسي
- `.gradient-secondary` - التدرج الثانوي
- `.gradient-company` - تدرج الشركات
- `.gradient-individual` - تدرج مستقلي HR
- `.gradient-employee` - تدرج الموظفين
- `.text-gradient-primary` - نص بتدرج لوني
- `.hover-lift` - تأثير رفع عند التمرير

## ملاحظات التصميم

- جميع البطاقات تستخدم `rounded-xl` للزوايا الناعمة
- الظلال: `shadow-card` للبطاقات
- التأثيرات: `hover-lift` للعناصر التفاعلية
- دعم كامل للغتين العربية والإنجليزية مع تغيير الاتجاه تلقائياً
